### 1.2 颜色系统

- **主蓝色**：`#0072F5` (rgb(0, 114, 245))

- **橙色**：`#FF7B00` (rgb(255, 123, 0))
  - 用于"搜索"按钮背景

- **绿色**：`#5FCA96` (rgb(95, 202, 150))
  - 用于影响因子（Impact Factor）标签
  - 如：`[IF:16.6]`, `[IF:33.2]` 等

#### 背景色
- **主背景**：`#FFFFFF` (白色)
- **次级背景**：`rgb(245, 247, 252)` - 浅蓝灰色
- **卡片背景**：`rgb(242, 242, 242)` - 浅灰色

#### 文本颜色
- **主文本**：`rgb(51, 51, 51)` (#333333)
- **次要文本**：`rgb(128, 128, 128)` (#808080)

### 1.3 核心组件样式

#### "头条"标签
- 背景：浅灰色
- 文字：深灰色
- 圆角：小圆角
- 内边距：紧凑型

#### 搜索组件
```
结构：[输入框] + [搜索按钮]

  - 边框：白色/浅色
```

   - 用途：CTA按钮、搜索按钮等关键操作
   - 理由：参考网页使用橙色作为强调色，形成蓝橙配色方案

   - 颜色值：`#5FCA96`

2. **微调背景色**

   - 卡片背景：从 `#F5F5F5` 调整为 `rgb(242, 242, 242)`

3. **扩展字体家族**
   - 添加 `Roboto` 以提升 Android 设备显示效果

### 5.3 低优先级改进 🟢

1. **实现日期卡片组件**（可选）
   - 蓝色背景 + 白色文字
   - 用于日期/编号展示
   - 参考首页的日报列表样式

2. **添加组件级样式令牌**
   - 作者标签圆角：`17px`
   - 标签内边距：`0px 10px`

## 七、设计系统总体评价

### 当前项目优势 ✅
1. ✅ 主蓝色完美匹配参考网页（#0072F5）
2. ✅ 字号层级体系完整（h1-h5, body, caption等）
3. ✅ 间距系统覆盖全面（4px-32px）
4. ✅ 支持亮色/暗黑双主题切换
5. ✅ 使用 vw 单位实现真正的响应式布局
6. ✅ 基于 Ant Design Mobile 提供完整组件库

### 需要改进的点 ⚠️
1. 🔴 **缺少橙色强调色** `#FF7B00`（重要新发现！）
   - 参考网页用于搜索按钮、关键操作
   - 形成蓝橙配色方案，提升视觉对比
2. ⚠️ 正文字号偏小（14px vs 16px）
3. ⚠️ 缺少行高令牌
4. ⚠️ 次要文本颜色略浅（#666 vs #808）
5. ⚠️ 背景色略有差异

### 核心发现 💡
通过分析**首页**和**文章详情页**两个页面，发现参考网站采用：
- **蓝橙绿三色配色方案**：
  - 主蓝色 `#0072F5` - 主色调，用于链接、日期卡片
  - 橙色 `#FF7B00` - 强调色，用于CTA按钮
  - 绿色 `#5FCA96` - 成功色，用于影响因子标签
- **16px正文字号** + 26px行高（1.625倍）
- **特色组件**：日期卡片、影响因子标签、搜索框组合

### 整体结论 ⭐⭐⭐⭐
当前项目的设计系统已经非常接近参考网页，**主蓝色完美匹配**，字号和间距体系完整。

**最重要的发现**：参考网站采用**蓝橙绿三色配色方案**
- 🔵 蓝色 `#0072F5` - 主色调（已匹配✅）
- 🟠 橙色 `#FF7B00` - 强调色（**需添加**）
- 🟢 绿色 `#5FCA96` - 成功色（**需添加**）

这个三色方案形成完整的视觉层次：蓝色建立品牌认知，橙色引导关键操作，绿色传达正向价值。配合正文字号调整（16px）和行高令牌，即可达到参考网页的视觉效果。
